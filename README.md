## About
基于 Vue3 (vite) 搭建树莓派桌面小屏应用前端

## Feature

* 提供彩云天气信息
* 提供树莓派系统信息，如CPU、内存、网路
* 提供todo信息

## Usage
```
# 执行
pnpm install
pnpm run dev
```
